# C Language

The C language is has a very rich ecosystem owing to it's long history. It's
*the* language when it comes to working on low-level systems—be it operating
systems, microcontroller, IoT devices, routers, or any other embedded devices.
The reason is simple: it's the simplest possible abstraction over the raw
instructions fed to the controlled (assembly).

## Learning It

> With great tools come great ecosystem. — Anonymous

Learning the basic syntax takes <10 minutes
[here](https://learnxinyminutes.com/c).
Understanding it however is a different game. It takes a lot of time and
effort.

## Idea

- Learn something new each day (or as often as possible).
- Understand it deeply. Comment in source.
